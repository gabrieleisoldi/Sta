Chi ha inventato questo linguaggio?

Il linguaggio Html è stato inventato da Tom Berners-lee mentre lavorava al CERN di Ginevra.


Quando è stato inventato?

Venne inventato nel 1989


Quali sono gli scopi di questo linguaggio?

Lo scopo pincipale del linguaggio html è descrivere la struttura e il contenuto delle pagine web utilizzando dei tag.


Che cose il WEB?

Il web è un sistema di documenti interconessi tramite link online, accessibili attraverso internet.


Che problemi risolveva il WEB?

Ha risolto problemi come l'accesso all'informazione, la comunicazione, il commercio, la collaborazione e l'intrattenimento.


Che differenza tra web e html ed internet?

Internet è una rete globale di computer interconnessi che utilizzano dei protocolli per comunicare, il web è un servizio che usa internet per accedere a documenti tramite il protocollo http mentre html un linguaggio informatico per creare le strutture delle pagine web.
